# HTML(수학교재) → 한글(HWPX) 변환기

mathbook 양식의 **HTML 교재**(MathJax 수식 + 경기천년 폰트 + base64 이미지)를
**편집 가능한 한글 문서(.hwpx)**로 자동 변환하는 도구.

수식은 한글 **네이티브 수식 개체**(더블클릭 편집 가능)로, 박스는 한글 **표(셀)**로,
원본 글꼴·크기·간격을 그대로 재현한다.

---

## 1. 빠른 사용법

### 방법 A — 드래그&드롭 (가장 쉬움)
1. 변환할 `*.html` 파일을 **`변환하기.bat` 위로 끌어다 놓기**
2. 끝. HTML과 **같은 폴더**에 `<이름>_한글변환_<날짜_시간>.hwpx` + 미리보기 `.pdf` 생성

### 방법 B — 더블클릭
1. `변환하기.bat` 더블클릭 → 파일 선택창에서 HTML 고르기

### 방법 C — 명령줄 (옵션 조절)
```bat
python engine\build.py "C:\경로\개념.html"
python engine\build.py "개념.html" --no-pdf            # PDF 미리보기 생략
python engine\build.py "개념.html" --out "결과.hwpx"   # 출력 경로 지정
python engine\build.py "개념.html" --start 0 --end 2   # 앞 2개 시트만(샘플 변환)
```

> 변환 중 한글 창이 잠깐 떴다 사라질 수 있다. **닫지 말 것**(자동으로 처리됨).

## 2. 설치 요건 (이 PC엔 이미 다 됨)

| 항목 | 비고 |
|---|---|
| **한컴오피스 한글** | COM 자동화 사용 (한글 2020 이상 권장) |
| **Python 3.x** | PATH에 등록되어 있어야 함 |
| Python 패키지 | `pywin32`, `beautifulsoup4`, `lxml` |
| **경기천년 폰트** | 경기천년제목(Medium/Bold), 경기천년바탕(Regular/Bold) — 무료 공개폰트 |

패키지 설치(필요 시):
```bat
python -m pip install pywin32 beautifulsoup4 lxml
```

## 3. 무엇이 어떻게 변환되나

| 원본(HTML) | 결과(한글) |
|---|---|
| 인라인 수식 `$...$`, 디스플레이 `$$...$$` | **네이티브 한글 수식 개체** (더블클릭 편집) |
| 표지/제목/소단원 계층 | 글꼴·크기·굵기 반영한 문단 |
| 공식 박스 · 표지목차 박스 · 제시문 박스 | **한글 1칸 표**(테두리/여백 GUI 편집 가능) |
| 채점기준 박스 | **항목별 칸(내용·점수) 표** + 제목칸 병합 |
| 그래프 이미지(base64) | 문서에 포함된 그림 |
| 글꼴 | 제목=경기천년제목, 본문=경기천년바탕 |

### 수식 표기 규칙 (정자체 / 이탤릭)
한글 수식 엔진의 자동 처리에 맡겨, 다음이 자연스럽게 적용된다.
- **정자체**: 함수명 `sin cos tan log lim …`, 대문자 점·선분 라벨 `\mathrm{AB}`
- **이탤릭**: 변수 `a,b,c,t,x`, 함수기호 `f,g`, 각 변수, 그리스문자 `α,β,γ,θ`
- 예) `\tan t` → **tan**(정자체) *t*(이탤릭),  `f(g(x))` → 전부 이탤릭

## 4. 직접 편집 / 커스터마이징

### 한글에서 박스 편집
박스는 전부 **표**다. 박스 안 클릭 → 셀 선택 후:
- 테두리 두께·색: 우클릭 → **셀 테두리/배경**
- 안쪽 여백: 우클릭 → **표/셀 속성** → 안 여백
- 행 추가/삭제: 표 안 우클릭 → 줄 추가/지우기

### 변환 기본값 바꾸기 — `engine/build.py` 상단
```python
F_TITLE_B = "경기천년제목 Bold"   # 글꼴 바꾸려면 여기
F_BAT_R   = "경기천년바탕 Regular"

BOX = {  # 박스 테두리 두께/색
    "formula": dict(color=(0,0,0), width="0.4mm"),
    ...
}
STYLE = {  # 스타일별 글자크기(pt*100)·줄간격(%)·문단간격(mm)
    "h1":   dict(face=F_TITLE_B, size=1800, lh=135, before=0, after=3.5, align="left"),
    "body": dict(face=F_BAT_R,   size=1050, lh=165, before=0, after=0.8, align="justify"),
    ...
}
```
숫자만 바꾸고 다시 변환하면 반영된다. (선 두께는 `0.1,0.12,0.15,0.2,0.25,0.3,0.4,0.5,0.7,1.0mm` 중 선택)

## 5. 한계 / 주의

- **mathbook 양식 전용**: `.sheet / .formula / .stmt-box / .grading-box / .concept …`
  클래스 구조를 가정한다. 다른 구조의 HTML은 본문은 들어가도 박스/스타일이 안 맞을 수 있다.
- **폰트 임베드 불가(자동화)**: 다른 PC에서 똑같이 보이려면 그 PC에 경기천년 폰트를 설치하거나,
  한글에서 **[다른 이름으로 저장] → "문서에 글꼴 포함"** 체크.
- 레이아웃은 한글 엔진이 새로 흘려보내므로 원본과 페이지 나눔이 다를 수 있다(내용·수식은 보존).

## 6. 폴더 구성
```
HTML을_한글로_변환기/
├─ 변환하기.bat          ← 드래그&드롭 / 더블클릭 실행
├─ README.md            ← 이 문서
└─ engine/
   ├─ build.py   ← 메인 변환기(HTML 파싱 + 한글 COM 빌드)
   └─ conv.py    ← LaTeX → 한글 수식 스크립트 변환 엔진(자체 구현)
```

LaTeX↔한글수식 매핑 개념은 공개 프로젝트
[`soseon203/exam-paper-korean`](https://github.com/soseon203/exam-paper-korean)을
참고하여 자체적으로 재작성하고, 축약형 분수/루트·도(°)·정자체 규칙 등을 보강했다.
